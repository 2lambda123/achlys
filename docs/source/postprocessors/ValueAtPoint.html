ValueAtPoint

buildconstruction:Undocumented Class

The ValueAtPoint has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

Find a specific target value along a sampling line. The variable values along the line should change monotonically. The target value is searched using a bisection algorithm.
