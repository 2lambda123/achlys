DirichletPfcFunctionFluxRampBC

buildconstruction:Undocumented Class

The DirichletPfcFunctionFluxRampBC has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

Imposes the essential boundary condition , where is a constant calculated assuming the recombination fluxquickly becomes equal to the incident flux. The recombination term is calculated using the simulation temperature and the pre-exponent and Energy values provided
