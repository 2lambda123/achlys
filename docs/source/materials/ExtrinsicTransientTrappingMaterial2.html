ExtrinsicTransientTrappingMaterial2

Overview

This class implements the trapping material class for the case where additional extrinsic trapping sites are created over time due to the influence of heavy particle bombardment.

Note that once a calculation is complete using this class for the creation of new traps, subsequent calculations can use one of the complementary classes which will maintain this final state.

  • ExtrinsicStaticTrappingMaterial for constant temperature cases

  • ExtrinsicStaticTrappingMaterialRamptingT for cases with a constant temperature ramp, for example in the case of thermal desorption
